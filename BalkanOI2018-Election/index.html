[BalkanOI2018] Election

发表于 2023-10-07 分类于 题解 ， 其他 阅读次数：

今天又是降智的一天。

题目链接

题意

给定一个长度为 $ n $ 的字符串 $ S $ ，由 C 和 T 两种字符构成。

$ q $ 次询问，每一次询问给出 $ l,r $ ，要求在字符串 $ S[l,r] $ 内删除若干字符。

使得对于该子串的任何一个前缀、后缀，都有 C 的数量大于 T 的数量。

求出最小删除字符数。

$ 1\le n,q\le 5\times 10^5 $

题解

首先经典的题意转化，将 C 视作 $ 1 $ ，T 视作 $ -1 $ ，则要求所有前后缀和 $ \ge 0 $ 。

那么我们删除的只可能是 $ -1 $ ，而不可能是 $ 1 $ 。

首先来想，假如只考虑前缀和 $ \ge 0 $ 该怎么做呢？

显然，最小个数一定是 $ -\min\limits_{i=0}^n pre_i $ ，因为对于每一个位置 $ i $ ，只有删除至少 $ -pre_i $ 个数字，才能使前缀和 $ \ge 0 $ 。

考虑构造这个删除方案，直接找到第一个前缀和为 $ -1,-2,\dots $ 的位置，并把他们删掉即可，这个方案显然是合法的。

再来考虑后缀和，我们发现，最优的方案，一定是正着做一遍上面的构造，再反着做一遍上面的构造。

我们在正着的构造中删除数字的时候，一定删除的是能删的”最靠后”的位置，这样可以尽可能地增大更多的后缀和。正着做完之后，我们就把问题转化成了只考虑后缀和的子问题，所以再反着做一遍就好了。

设正着做完后，第 $ i $ 个位置的后缀和是 $ suf’_i $ 。

那么我们要求的答案就是 $ -\min\limits_{i=0}^npre_i-\min\limits_{i=1}^{n+1}suf’_i $ 。

考虑 $ suf’_i $ 的值是多少，一定有 $ suf’_i=suf_i+c $ ，其中 $ c $ 为正着构造时，在 $ [i,n] $ 内删除的数字个数。

那么就会有 $ c=-\min\limits_{j=0}^npre_j+\min\limits_{j=0}^{i-1}pre_j $ ，其中 $-\min\limits_{i=0}^npre_i$ 为删除的数字总数，$ -\min\limits_{i=0}^{i-1}pre_i $ 是在区间 $ [0,i) $ 内删除的数字个数，两者做差即为 $ c $ 。

所以 $ suf’_i=suf_i-\min\limits_{j=0}^npre_j+\min\limits_{j=0}^{i-1}pre_j $ 。

则答案为 $ -\min\limits_{i=0}^npre_i-\min\limits_{i=1}^{n+1}\{suf_i-\min\limits_{j=0}^npre_j+\min\limits_{j=0}^{i-1}pre_j\}=-\min\limits_{i=1}^{n+1}\{suf_i+\min\limits_{j=0}^{i-1}pre_j\}=-\min\limits_{i=1}^{n+1}\min\limits_{j=0}^{i-1}(suf_i+pre_j) $

也就是说，我们需要找到一段前缀与一段后缀，使得他们不相交，且和最小(答案为它们的相反数)。

即答案为区间最大子段和-区间和，线段树维护即可。

时间复杂度为 $ O(n+m\log n) $ 。
